What operating system do you use? (#what-os)

	
	I use both Artix Linux (a variant of Arch Linux), and Gentoo Linux.

	Regarding the kernel, I use linux-hardened when all I want is security. This uses the default package from Arch's repositories, and has not been customized.

	Also, I use my own build of linux-ck, that does not contain useless things (AMD, nVidia, etc.),
	and is useful when I want my laptop (ThinkPad T440p) to give me the best performance possible.
	Of course, I modified this kernel:

		Before modifying it (stock settings, only required modules using modprobed-db), it took around 30 minutes to compile and install!

		After modifying it (other unnecessary modules removed, CPU & GPU number lowered to match my system config, same modprobed-db database file),
		it only takes around 7 minutes to compile and install! Also, the performance is far better than linux-hardened.
