What OS do you use? Why?

I use Ubuntu 18.04 LTS x64 on a Lenovo B50-10 laptop. I use Ubuntu specifically because:
1) It's updated regularly; 2) It's debian based; 3) It's really easy to work with it; 4) I like how I can customize it.

Do you think that Windows is better than Linux?

I think that, whichever flavor of Linux you use, from OpenBSD, to Red Hat Linux, you're still a Linux user, and nobody can make fun of you, because Windows is inferior than Linux.
From a security point of view, Windows definitely has a lot more vulnerabilities than Linux, plus, it's closed source, while, on the other hand, Linux is open source, which means that anybody can contribute to improve the Linux kernel.
Instead, from a gaming point of view, Windows has more games developed for it, but that doesn't mean it's better, since you can also run windows executables (.exe) on Linux, via WINE, PlayOnLinux, etc.

Also, I talked about being really good at speaking English: why does that matter? Well, it matters because, without being able to speak/understand English, I wouldn't be here writing this.
I speak Italian, English and French. English is a really important language if you want to speak to anyone, honestly. Computer programming also requires quite a bit of English knowledge.
