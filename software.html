My Software

I have wrote some programs in C. Here's a list down below:
  • Csnake: a snake clone written in C
  • Quiz: create a Quiz, and/or take a quiz
  • Calc: a simple calculator written in C, with support for some special values, bit-shifting, Reverse Polish Notation, etc.

All of the projects above use GNU AutoTools for the building process, to increase portability.

Additionally, I have also written my own C Library, called SMCLIB. At the moment, SMCLib contains the following functions:

  • color(char *str, int colorCount, ...): color a string. Example: 
    /* Make the string below bold, with a red background */
char *myStr = "Hello, World!";
myStr = color(myStr, 2, bold, redBg);
  • _free_color(void): free a colored string (it was malloc'd). If you're using a variable, just free() that instead, otherwise use the function above.

Another feature of this Library is detection of which compiler was used to compile the program, and which platform was used to compile/which architecture was targeted.

To access the compiler info, look into the contents of CC. Similarly, for the architecture info, look into the contents of ARCH.
